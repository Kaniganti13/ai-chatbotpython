# Aurora AI Chat — Minimal AI Chatbot (No DB)

A simple AI chatbot web application built with **FastAPI** and plain **HTML/CSS/JS**. No database required — chat history is stored in the browser.

## Features

- Landing page with animated illustration
- Chat UI with dark mode, glassmorphism, and typing animation
- Send messages to OpenAI Chat API
- No database — stateless backend
- Clear chat, auto-scroll, Enter to send, responsive design

## Prerequisites

- Python 3.10+
- OpenAI API key (or swap to Gemini API if you prefer)
- (Optional) Node not required — frontend is static
