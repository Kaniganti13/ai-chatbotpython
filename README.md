# Aurora AI Chat ✨

Aurora AI Chat is a modern AI-powered chatbot web application built using FastAPI, HTML, CSS, and JavaScript.  
It provides a clean conversational interface with real-time AI responses and a visually appealing UI.

---

## 🚀 Features

- 🤖 AI-powered chatbot
- 💬 Real-time conversation interface
- 🌙 Modern dark mode UI
- ✨ Glassmorphism design
- ⚡ FastAPI backend
- 📱 Responsive layout
- ⌨️ Typing animation effects
- 🔄 Auto-scrolling chat interface

---

## 🛠️ Technologies Used

- Python
- FastAPI
- HTML5
- CSS3
- JavaScript
- OpenAI / Gemini API

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/Kaniganti13/ai-chatbotpython.git
```

### 2. Navigate to Project Folder

```bash
cd ai-chatbot
```

### 3. Create Virtual Environment

```bash
python -m venv venv
```

### 4. Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 API Configuration

Create a `.env` file and add your API key:

```env
API_KEY=your_api_key_here
```

⚠️ Never upload API keys to GitHub.

---

## ▶️ Run the Project

```bash
python app.py
```

OR

```bash
uvicorn app:app --reload
```
